PSYCOPG2 Notes
pyscopg2.connect(dbname=, user=, password=) # connects to a database
cursor = connection.cursor() # sets cursor to a cursor item

cursor.execute('sql command goes inside this  string') # no ; at end of code, executes the sql code
connection.commit() # commits any changes made to the database.

connection.rollback() # rolls back partial changes made use to catch errors/prevent partial info going into db

cursor.executemany('SQL statement', data for statement)

default cursor returns data is a tuple of tuples
cursor = connection.cursor(cursor_factory=psycopg2.extras.DictCursor) creates a dictionary cursor.

use placeholders instead of directly writing values into statements, increase security and performance
